# Insider-Threat-SOC-Simulator

## Problem Statement
A legitimate internal user is suspected of misusing authorized access to
sensitive organizational resources. This project focuses on detecting,
investigating, and reporting insider threat activities.

## Objectives
- Detect abnormal user behavior
- Monitor file access and privilege usage
- Preserve forensic evidence
- Generate investigation reports

## Team Structure
All members contribute as Blue Team analysts with divided responsibilities.

## Tech Scope
- Log analysis
- Rule-based detection
- Forensics & reporting
